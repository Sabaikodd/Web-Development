List in HTMl:
     while making list in HTMl list tag is used <li> //body</li>.
     But alone list tag is not enough.There are 2 type of list:
         1.Unordered List:
                 -In this list bullet points are used to make any list no element in list are in order.
                 - <ul>
                       <li>//body</li>
                   </ul>
     Eg: This way a small unorderlist will be created
        <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>List in HTML</title>
    </head>
    <body>
        <ul>
            <li><h1>Things to buy</h1></li>
            <li>Bread</li>
            <li>Jam</li>
            <li>Butter</li>
        </ul>
    </body>
</html>
           2.Ordered list:
                  -In this list 1.2.3.or A.B.C. are used to make any list all element in list are in order.
                  - <ol>
                       <li>//body</li>
                   </ol>
     Eg: This way a small orderlist will be created

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>ordered  list in HTML</title>
    </head>
    <body>
       <ol>
        <li><h1>Things to buy</h1></li>
            <li>Bread</li>
            <li>Jam</li>
            <li>Butter</li>
       </ol>
    </body>
</html>
Or you can also add type="A" attribute inside orderedlist starting tag like <ol type="A"> to get A.B.c. in output.

 # HTML attributes  are used to add more info,value,text to the tag like we did above.
