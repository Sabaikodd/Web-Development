Inline and block elements
     Inline elements : 
                -are those elememts that takes necessary width ans space for themself.
                -doesnot start from new line.
                -eg:Image element,anchor element.
     Block elements:
               - are those elements that takes up whole block/width even if they dont need it.
               - starts from new line.
               -eg:paragraph element,heading element.
      
       Eg of block element:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Inline v/s Block</title>
</head>
<body>
    <h1>Inline element</h1>
    <h1>Inline element 2</h1>   
    <p> This is also</p>
</body>
</html>
